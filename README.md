# Operations Workflow Showcase

本頁為個人作品展示，整理我在電商倉儲與營運支援工作中，針對進貨、出貨、庫存紀錄與資料核對流程所進行的改善專案。

此作品主要呈現我如何觀察日常作業中的重複性問題，將原本仰賴人工判讀、手寫紀錄與多份表單交叉核對的流程，整理成更清楚、可追蹤、具防呆機制的作業方式。

## Project Overview

此專案以「提升行政營運效率」與「降低人工作業錯誤」為核心，針對進出貨管理流程進行整理與優化。

主要改善方向包括：

- 進貨資料紀錄與整理
- 出貨訂單核對
- 庫存異動追蹤
- 商品資料標準化
- 作業流程防呆設計
- 週期性資料統計與報表支援
- SOP 文件化與流程梳理

## Key Features

- 將分散於多份表單與人工紀錄中的資料，整理為更容易追蹤的流程
- 透過欄位設計與核對邏輯，降低人工輸入與判讀錯誤
- 協助現場作業人員更快速完成進貨、出貨與庫存確認
- 支援主管查看週期性資料與作業結果
- 將日常營運經驗轉化為可展示、可說明、可延伸的作品

## Role & Contribution

我在此專案中負責：

- 觀察既有作業流程與痛點
- 整理進貨、出貨、庫存與報表需求
- 設計資料欄位與作業邏輯
- 建立防呆與核對機制
- 製作展示頁面與專案說明
- 將實務經驗轉化為履歷作品集內容

此專案並非單純的技術練習，而是來自實際工作情境中的流程改善經驗。

## Skills Demonstrated

此作品呈現以下能力：

- 行政營運流程整理
- 資料管理與核對
- SOP 建立與文件化
- 問題分析與改善提案
- 跨角色溝通與需求整理
- 自主學習與工具應用
- 細節控管與錯誤預防
